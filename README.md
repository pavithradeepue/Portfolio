# 🌸 Pavithra Deepu E - Personal Portfolio

![Portfolio Preview](./public/favicon.png)

Welcome to my personal, interactive web portfolio! This repository contains the source code for my portfolio website, designed to showcase my journey, projects, skills, and academic achievements in a sleek, modern, and highly interactive manner.

## ✨ Features

- **Modern UI/UX**: Designed with a lavender-pink glowing theme utilizing dynamic "glass-morphism" cards and smooth CSS transitions.
- **Fully Responsive**: Optimised effectively for both desktop and mobile viewing.
- **Rich Data Rendering**: Everything from my experience timeline to my achievements and community engagements is dynamically rendered from `src/data/portfolio.ts`.
- **Certificate Downloads**: Clicking on any of my certifications or event participation cards instantly enables you to download the respective `.pdf` or image proof seamlessly!
- **Sleek Animations**: Custom keyframe animations (like my blinking link hover effects) bring the page to life.

## 🛠️ Built With

- **[React](https://reactjs.org/)** (v18)
- **[TypeScript](https://www.typescriptlang.org/)** - For type-safe component and data structuring
- **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first styling with custom theme layer implementations
- **[Lucide React](https://lucide.dev/)** - Beautiful & consistent iconography
- **[Framer Motion](https://www.framer.com/motion/)** - For scroll-based layout animations

## ⚙️ Running Locally

Want to test it out on your local machine? Ensure you have Node.js installed.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pavithradeepue/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Visit the app** at `http://localhost:8080` (or whichever port Vite maps).

## 🗂️ Project Structure

- `src/data/portfolio.ts` - 🧠 **The Brain**: Edit this single file to update my skills, experience, certifications, and achievements!
- `src/components/` - Holds all the modular UI puzzle pieces (e.g., `HeroSection.tsx`, `CertificationsSection.tsx`).
- `src/index.css` - Contains the global CSS utility layers and custom `@keyframes`.
- `public/` - Houses all my downloadable certificate PDFs, resume, and images.
